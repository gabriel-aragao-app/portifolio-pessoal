Como funciona o CSS Grid?

Exemplo Visual e Prático

Item 1 (Cabeçalho):
Este item usa grid-column: span 3. Ele se estica por todas as 3 colunas do container pai, funcionando como um topo perfeito para o layout.
Item 2 (Coluna 1):
Ocupa naturalmente o espaço de 1 fração (1fr) definido na estrutura principal do CSS.
Item 3 (Coluna 2):
O Grid calcula o espaço automaticamente e distribui os itens em ordem de aparição no código.
Item 4 (Coluna 3):
Como a primeira linha já foi preenchida pelo Item 1, os itens 2, 3 e 4 se alinham lado a lado na segunda linha.
Item 5 (Bloco Duplo):
Este bloco usa grid-column: span 2. Ele pula para a linha de baixo e consome o espaço equivalente a duas colunas juntas.
Item 6 (Rodapé):
Aqui usamos grid-column: 1 / 4. Isso diz explicitamente para o elemento começar na linha de grade 1 e terminar na linha de grade 4, cobrindo o layout de ponta a ponta.
